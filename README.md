# A股股价查询插件

一个用于查询A股股票实时股价的 AstrBot 插件，数据来源于新浪财经。

## 功能

- 查询沪市、深市主板、创业板股票实时股价
- 支持纯数字代码和带前缀代码两种格式
- 显示当前价格、涨跌幅信息

## 使用方法

```
/查询股价 [股票代码]
```

### 支持的格式

- **纯数字格式**：`600000`、`000001`、`300001`
- **带前缀格式**：`sh600000`、`sz000001`、`sz300001`

### 示例

```
/查询股价 600000      # 查询浦发银行
/查询股价 sh600000    # 查询浦发银行（带前缀）
/查询股价 000001      # 查询平安银行
/查询股价 sz300001    # 查询特锐德（创业板）
```

## 数据说明

- 当前价：股票最新成交价
- 涨跌幅：相对于昨日收盘价的涨跌幅度
- 数据来源：新浪财经

## 注意事项

- 仅支持A股股票查询
- 股票代码为6位数字
- 沪市代码以6开头，深市主板以0开头，创业板以3开头
